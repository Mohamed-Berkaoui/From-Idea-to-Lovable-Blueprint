üë§ What is a User Definition?

A user definition is a clear description of the person who will use your app. It includes:

  • Who they are: Age, job, lifestyle
  • Their situation: What's happening in their life
  • Their problem: What frustrates them
  • Their goal: What they want to achieve

üéØ Single User vs Multiple Users

Some apps have ONE type of user. Others have MULTIPLE types:

Single User Apps

  • Weather App ‚Üí One user checking weather
  • Calculator ‚Üí One user doing math
  • Habit Tracker ‚Üí One user tracking habits
  • Notes App ‚Üí One user writing notes
  • Timer App ‚Üí One user timing activities

Multi-User Apps

  • E-commerce ‚Üí Customer + Admin
  • Uber ‚Üí Rider + Driver
  • Airbnb ‚Üí Guest + Host
  • School App ‚Üí Student + Teacher + Parent
  • Food Delivery ‚Üí Customer + Restaurant + Driver
üí°

Important: For multi-user apps, define EACH user type separately! Each user has different goals and actions.
