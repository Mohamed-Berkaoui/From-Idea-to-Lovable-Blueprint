üìù The Prompt Structure

A complete AI prompt has these parts:

1

USERS

Who uses the app (can be multiple)

2

GOALS

What each user wants to achieve

3

ACTIONS

What each user can do (5-7 per user)

4

FLOWS

Main journeys through the app

5

SCREENS

Wireframes for key screens

üéØ Tips for Better Prompts

  • Be specific: The more detail, the better the result
  • Use simple language: AI understands plain English best
  • Organize clearly: Use headers and bullet points
  • Include all user types: Don't forget admins/managers
  • Describe screens: AI can visualize from text descriptions

Common Mistake: "Make me an app like Uber" - This is too vague! Break it down into specific details.

An online shopping app where customers can browse and buy products, and store owners can manage their inventory and orders.

üë• USERS

  8. Write reviews
‚öôÔ∏è ADMIN ACTIONS

  5. Update order status
  6. View sales reports
  7. Manage customer accounts

‚úÖ Pre-Prompt Checklist

Before sending your prompt to AI, make sure you have:

  • Described ALL user types
  • Written clear goals for EACH user
  • Listed 5-7 actions for EACH user
  • Described at least ONE complete flow
  • Listed all screens needed
  • Included wireframes for key screens (optional but helpful)
üí°

Remember: The more detail you provide, the better the AI can help you build exactly what you need!
